### Problem
https://codeforces.com/contest/2149/problem/B
<details>
  <summary>Observation</summary>
You have to make the difference between all pairs minimum. Try to use greedy!
</details>
<details>
  <summary>Solution</summary>
Sort all elements in array $a$ then just check the maximum value of every two adjacent numbers
</details>

### Time Complexity
O(N log N)
